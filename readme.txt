3. All Spring and Hibernate jar files stored in spring+hibernate jar files, add it to project inside /Webcontent/WEB-INF/lib folder.
4. Install Tomcat server and deploy this project inside it.
5. Run this application using http://localhost:8080/LibraryManagementSystem/.
